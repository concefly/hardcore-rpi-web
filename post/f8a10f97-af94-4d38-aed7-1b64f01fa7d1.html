安装 kodi

kodi 是一个屡获殊荣的自由和开源的跨平台媒体播放器和用于 HTPC（Home theater PC，家庭影院PC）的数字媒体娱乐中心软件，可以运行在 Linux、OSX、Windows、Android 系统。最初为 Xbox 而开发，叫 XBMC(全称是XBOX Media Center)，顾名思义就是 Xbox 平台的媒体中心。

更多关于 kodi 的介绍可以看它的官网：http://kodi.tv/about/xbmc-foundation

这一小节就教大家把 kodi 安装到树莓派上。

此部分内容需要一点点的命令行基础，并能通过 ssh 连接登录树莓派

首先 ssh 登录树莓派，安装 kodi 软件包：

sudo apt-get update
sudo apt-get install kodi

执行安装之前，强烈建议先给树莓派更新软件源，否则软件包下载速度会非常慢：树莓派更换软件源

更多的官方安装教程可以参考：

  • https://kodi.wiki/view/HOW-TO:Install_Kodi_on_Raspberry_Pi

首次启动 kodi

我们先简单启动下 kodi，验证是否能正常运行。

验证方式非常简单，按如下步骤进行：

  1. 树莓派 HDMI 连接上显示器，并通电启动
  2. ssh 登录树莓派
  3. 命令行输入 kodi 启动

命令执行后，显示器就会出现 kodi 的界面，说明 kodi 已经成功运行起来了！
